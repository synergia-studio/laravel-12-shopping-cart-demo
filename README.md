# laravel-12-shopping-cart-demo
Simple shopping cart done with Laravel PHP framework.

Edit in site/.env:

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laravel-12-shopping-cart-demo
DB_USERNAME=root
DB_PASSWORD=rakics98

Import MySql database dump: laravel-12-shopping-cart-demo.sql

From DOS prompt go to: 

>cd site

run server:

>php artisan serve

Open site: http://localhost:8000/login/
